About Us

Welcome to MMTR! These days, keeping track of tasks can be difficult, so I've made a simple and intuitive platform to help you stay organized and on top of your daily to-do list. This web app allows you to easily create, edit, and delete tasks, set reminders for important deadlines, and prioritize your tasks based on their importance.

This application is built using the latest web technologies, including Angular, Firebase, and Bootstrap. This ensures that this application is fast, reliable, and scalable.

The main goal is to make task management a stress-free experience, so you can focus on what's important. Kindly note that this application is still under development. So, Please inform us if you find any difficulties using it. The Notification mechanism is still under development.
